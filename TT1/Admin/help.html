User Manual  

 

1. Administrator Login

Figure_ below shows the administrator’s log in page, the user should log in to proceed to the home page. User should have username and password.
To log, in please follow the steps:
1.  Enter the username and password.
2.  Click the “Login” button. The system will validate the username and password.
3.  If login is valid, you will be redirected to the home page.
4.  If login failed, an error message will be displayed, and you have to reenter the username or   
     password.
